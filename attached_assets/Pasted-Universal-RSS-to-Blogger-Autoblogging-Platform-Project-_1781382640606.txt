Universal RSS-to-Blogger Autoblogging Platform

Project Overview

Build a modern SaaS web application that automatically imports content from RSS feeds, rewrites it using Google Gemini AI, publishes it to Blogger.com, and immediately distributes it to social platforms.

The platform should focus on simplicity, speed, and reliability.

No post scheduling system.

No queue system.

No delayed publishing.

Workflow:

RSS Feed → AI Rewrite → Blogger Publish → Social Distribution

All actions happen immediately after processing.

---

Core Goal

Create a complete autoblogging platform for affiliate marketing, crypto, personal finance, travel, pets, and other content niches.

The system must support:

- Multiple Blogger blogs
- Multiple RSS feeds
- Google Gemini AI rewriting
- Auto publishing to Blogger
- Auto sharing to Bluesky
- Auto sharing to Mastodon
- Auto sharing to Tumblr
- Auto sharing to Pixelfed
- Full admin dashboard
- SaaS-ready architecture

RSS Processing

When user clicks "Fetch Feed":

1. Fetch RSS XML

2. Parse content

3. Extract:
   
   - Title
   - Content
   - Image
   - URL
   - Author
   - Date

4. Check duplicates

Duplicate logic:

Do not process if source URL already exists.
